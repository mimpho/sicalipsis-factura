# Sicalipsis Factura 📄✨

**Sicalipsis Factura** es un generador de facturas minimalista, rápido y elegante diseñado específicamente para profesionales que valoran la simplicidad y el diseño impecable.

## ✨ Características principales

- **Diseño Premium**: Interfaz limpia y tipografía cuidada para una apariencia profesional.
- **Exportación a PDF**: Generación de documentos PDF listos para enviar con un solo clic.
- **Soporte Veri*factu**: Generación automática de códigos QR para cumplimiento normativo.
- **Multi-idioma**: Soporta Catalán y Castellano.
- **Responsive**: Totalmente adaptado para su uso en tablets y ordenadores.
- **Privacidad**: Todo el proceso se realiza en el navegador; no se guardan datos en ningún servidor externo.

## 🛠️ Tecnologías utilizadas

- **React 19**
- **Vite** (Build tool rápida)
- **Tailwind CSS** (Estilización)
- **Lucide Icons** (Iconografía minimalista)
- **html2canvas & jsPDF** (Motor de exportación PDF)
- **Motion** (Micro-animaciones fluidas)

## 🚀 Cómo empezar en local

1. **Clona el repositorio**:
   ```bash
   git clone https://github.com/mimpho/sicalipsis-factura.git
   cd sicalipsis-factura
   ```

2. **Instala las dependencias**:
   ```bash
   npm install
   ```

3. **Inicia el servidor de desarrollo**:
   ```bash
   npm run dev
   ```

## 🌐 Despliegue en GitHub Pages

Este proyecto está configurado para desplegarse fácilmente en GitHub Pages.

1. **Prepara el entorno** (si no lo has hecho):
   ```bash
   npm install --save-dev gh-pages
   ```

2. **Despliega con un comando**:
   ```bash
   npm run deploy
   ```

Asegúrate de configurar la fuente de GitHub Pages en los ajustes de tu repositorio a la rama `gh-pages`.

## 🎨 Personalización

Puedes ajustar los datos del emisor, logotipos y estilos visuales directamente en el código o mediante la interfaz de usuario en tiempo real.
